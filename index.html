ConOnRails by ConOnRails

An application for managing convention operations

Con On Rails is a successor to Thomas Keeley's "Con in a Box", the software that's been used to run CONVergence operations for many years. Major features include (or will include): * Tracking of current (and archiving of past) incidents being monitored by convention staff. * Tracking of lost-and-found. * Tracking of radio assets. * Tracking of volunteers and shifts. * A role-based authorization system for users. At present, it is not remotely feature complete, although some things do work. The goal is to have a feature complete version by March 2012 so that it can be thoroughly beaten up on before CONVergence 2012 in July. After that, the goal is to do some additional work to make it easy to generalize to other conventions.

Dependencies

Ruby 1.9.2 Rails 3.1 For other dependencies, see the Gemfile.

Install

Right now, this is not yet installable, per se, although it should run just fine as a rails app using 'rails server'.
